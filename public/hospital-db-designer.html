Entities and Attributes

Patient

  • Patient_ID (int, PK)
  • Name (string)
  • Age (int)
  • Gender (char)
  • Address (string)
  • Contact_No (string)

Doctor

  • Doctor_ID (int, PK)
  • Name (string)
  • Specialization (string)
  • Department_ID (int, FK)

Appointment

  • Appointment_ID (int, PK)
  • Patient_ID (int, FK)
  • Doctor_ID (int, FK)
  • Date (date)
  • Time (time)
  • Status (string)

Department

  • Department_ID (int, PK)
  • Name (string)
  • Location (string)
  • Head_Doctor_ID (int, FK)

Room

  • Room_ID (int, PK)
  • Room_Type (string)
  • Floor (int)
  • Status (string)

Nurse

  • Nurse_ID (int, PK)
  • Name (string)
  • Department_ID (int, FK)
  • Shift (string)
